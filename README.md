# Guía Módulo 2 — Desarrollo colaborativo con GitHub

Sitio web sencillo desarrollado para practicar el flujo de trabajo
colaborativo con Git y GitHub.

## Funcionalidades implementadas

- Estructura HTML semántica con `<header>`, `<main>` y `<footer>`
- Atributo `lang="es"` para accesibilidad (criterio WCAG 3.1.1)
- Hoja de estilos externa (`styles.css`) con tipografía y layout base
- Ancho de línea limitado a 700px para mejorar la legibilidad

## Flujo de trabajo aplicado

Cada cambio se desarrolló en una rama independiente, se propuso
mediante un pull request y se revisó con comentarios de línea antes
de fusionarse a `main`.

| Rama | Pull request | Cambios |
|---|---|---|
| `feature/estructura-semantica` | #1 | Etiquetas semánticas y accesibilidad |
| `feature/estilos-basicos` | #2 | Hoja de estilos externa |
